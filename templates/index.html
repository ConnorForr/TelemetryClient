API Endpoints

  • /car - POST endpoint for receiving JSON data from Arduino, requires authentication header.
  • /get/<date in YYYY-MM-DD format> - get data for a certain day.
  • /realtime/data - Get the most recent datapoints from the buffer.
  • /realtime/dummy - Get dummy datapoints generated by the server.
